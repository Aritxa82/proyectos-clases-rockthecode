Etiquetas de título

Las etiquetas de título son importantes en nuestra página para que semánticamente google pueda reconocer las partes más importantes de la misma y se crean con la etiqueta h (heading) seguida de un número, cuanto menor es el número mayor y más importante es el título y hay de límite hasta el número 6

<h1> Esto es el título principal </h1> <h2> Esto es el título segundo </h2> <h3> Esto es el título tercero </h3> <h4> Esto es el título cuarto </h4> <h5> Esto es el título quinto </h5> <h6> Esto es el título sexto </h6>
